The title The World of Beer Brew

Working with APIs, CSS Grid and Flex Box

See it Work

A Beer Brew page to search through Brewdog's expansive back catalogue of beer using the PUNK API.

The Web Application is used to display the beers from the Brewdog's catalogue. Contains filter to let you search the beer on the ABV level of alcohol or on the hoppines of the beer.

The data is displayed in card format using CSS Grid and Flexbox. On hovering over the card you can get a description of the beer along with the possible food choices that goes well with the beer.
